#CoreMedia Blueprint Extensions

##ES Demodata Generator

###Description

Elastic Social Demo Data Generator. The Elastic Social Demo Data Generator is a tool to simulate actions of a website 
online community.

It creates demo data like:

* Comments
* Users
* Reviews
* Ratings
* Blacklist entries
