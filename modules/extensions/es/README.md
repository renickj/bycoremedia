#CoreMedia Blueprint Extensions

##Elastic Social Extension

###Description

Extensions which integrates CoreMedia Elastic Social into CoreMedia Blueprint.

Wich features amongst others:

* Comments and Reviews
* User Profiles
* Moderation
* Registration
* Authentication
* User Management
* etc.
