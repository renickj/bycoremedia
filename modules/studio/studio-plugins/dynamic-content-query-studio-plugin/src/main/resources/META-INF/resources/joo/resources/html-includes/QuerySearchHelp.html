The search query is created via combinations of conditions. Only content that matches all your conditions will be displayed. Drag the condition to the query area. You can configure additional settings afterwards. Every condition can only applied once to the query area.

Condition Description
Taxonomies

There must be certain keywords or location in the documents.

Context

The context documents the queried content must belong to.

Modification Date

The documents must have been modified in a certain time range (e.g. the week before).
