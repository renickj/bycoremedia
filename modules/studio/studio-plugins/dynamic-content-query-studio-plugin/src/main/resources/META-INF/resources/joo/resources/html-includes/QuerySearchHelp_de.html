Über Bedingungen wird Ihre Suchanfrage zusammengestellt. Es werden nur Inhalte ausgegeben, die allen Ihren Bedingugen entsprechen. Ziehen Sie die gewünschte Bedingung in den Bereich der Suchanfrage. Danach können Sie dort weitere Einstellungen vornehmen. Jede Bedingung kann nur einmal innerhalb der Suchanfrage verwendet werden.

Bedingung Beschreibung
Taxonomien

Es müssen bestimmte Schlüsselwörter oder Verortungen in den Dokumenten vorhanden sein.

Kontext

Die Dokumente gehören zu einem oder mehreren Kontexten.

Änderungsdatum

Die Dokumente müssen in einem bestimmten Zeitraum geändert worden sein (z.B. eine Woche zuvor).
